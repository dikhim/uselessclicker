Basics


The computer is controlled by a script. To run the script, you must click the Start button on the toolbar, or press the CONTROL ALT A keyboard shortcut. To stop the script, you must click the Stop button or press the CONTROL ALT S keyboard shortcut.

Very important! Remember the CONTROL ALT S keyboard shortcut to stop the script. Often, this will be the only way to stop the hanging script.

You can also set your keyboard shortcut in the settings menu.

After pressing the start button, or a keyboard shortcut, the script is executed only once. Therefore, if you want to perform repetitive actions, you must use loops.
