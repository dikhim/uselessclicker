Version 0.4.0

Screen capturing

  • Added a screen object, with which you can capture an image from the display;
  • added tab Screen, where you can quickly select the part of the screen and save it.

Images

  • a new Image class that contains all the necessary methods for image processing;
  • searching for an image in another image using the findFirst, findAll, and findLimit methods;
  • images can be compiled by calling the method image.compile(), thereby consuming time for the compilation, but speed up further search;
  • added the image output view, where an image can be loaded, edited, saved to file or inserted to a script;
  • you can upload an image to the image output view from the script using the system.showImage() method.

Keyboard

  • Added new method key.typeText(String layout, String text) that typing the passed text in the specified layout;
  • the method key.typeText() supported layouts:
    "us" - English(US),
    "gb" - English(UK),
    "ru" - Russian,
    "ua" - Ukrainian.
  • Added the key ALT_GR for testing
