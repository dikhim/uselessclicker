Версия 0.4.0

Работа с экраном

  • добален объект screen, при помощи которого можно захватить изображение с дисплея;
  • добавлена вкладка Screen, где можно быстро выделить необходимую часть экрана и сохранить ее.

Работа с изображениями

  • новый класс Image, который содержит все необходимые методы для обработки изображения;
  • поиск изображения в другом изображении используя методы findFirst, findAll и findLimit;
  • изображения можно предварительно скомпилировать вызвав метод image.compile(), тем самым затратить время на компиляцию, но ускорить дальнейший поиск;
  • добавлено поле вывода изображения, где его можно предварительно отредактировать, сохранить, загрузить из файла либо из выбранной строки;
  • загрузить изображение в поле вывода из скрипта можно вызвав метод system.showImage();

Клавиатура

  • Добавлен новый метод key.typeText(String layout, String text) для набора текста в указанной раскладке;
  • для набора текста методом key.typeText добавлена поддержка раскладок:
    "us" - English(US),
    "gb" - English(UK),
    "ru" - Russian,
    "ua" - Ukrainian.
  • Добавлена клавиша ALT_GR для тестирования<./li>
