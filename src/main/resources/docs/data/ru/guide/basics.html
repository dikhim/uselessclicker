Основы


Управление компьютером выполняется посредством скрипта. Для того чтобы выполнить скрипт, необходимо нажать кнопку Запуск на панели инструментов, либо нажать сочетание клавишь CONTROL ALT A. Чтобы остановить скрипт, необходимо нажать кнопку Стоп либо сочетание клавишь CONTROL ALT S.

Очень важно! Запомните сочетание клавишь CONTROL ALT S для остановки скрипта. Часто, это будет единственный способ прекратить работу зависшего скрипта.

Вы также можете установить свои сочетание клавшиь в меню настроек.

После нажатия кнопки запуск, либо сочетания клавишь, скрипт выполняется только один раз. Потому, если вы хотите выполнить повторяющиеся действия, необходимо использовать циклы.
