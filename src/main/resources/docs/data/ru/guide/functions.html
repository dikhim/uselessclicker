Функции


Для создания функции добавим шаблон из меню Шаблоны>Язык>function echo. Данный пункт вставляет код для, так называемой, эхо-функции, которая принимает значение и возвращает его. 

function echo(text){
    return text;
}

Изменим функцию так, чтобы она принимала чиловое значение n и выводила в консоль значения от 0 до n. Для этого используем возрастающий цикл и системный метод system.println(). Чтобы вызвать созданную функцию, необходимо использовать конструкцию function(args);.

Скопируйте приведенный ниже код и запустите его. В консоль должны вестить значение от от 0 до 1 и от 0 до 2, для каждого вызова функции соответсвенно 

function count(n){
    for(i=0;i<n;i++){
        system.println('i='+i); 
    }
}
system.sleep(2000);
count(2);
system.sleep(2000);
count(3);

При запуске скрипта обратите внимание, что вывод в консоль производится не сразу, а с периодичность в две секунды. Здесь для наглядности используется метод system.sleep() который указывает, что системе следует подождать указанное количество миллисекунд. 2000 миллисекунд = 2 секунды.
