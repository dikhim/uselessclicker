Для работы с кнопками мыши и колеском есть встроенный объект "mouse". Методы для работы с кнопками аналогичны методам для работы с клавиатурой (press, release, perform, click). Посмотрите приведенный ниже код. 
mouse.press('LEFT');
mouse.release('LEFT');
        
mouse.button('RIGHT','PRESS');
mouse.button('RIGHT','RELEASE');

mouse.click('MIDDLE');

Также, у объекта "mouse" есть методы для работы с колисиком мыши. Для управления колесиком мыши необходимо указывать два параметра: направление прокрутки"UP/DOWN"и числовой параметр количество строк. 

mouse.wheel('DOWN',3);
mouse.wheel('UP',3);
