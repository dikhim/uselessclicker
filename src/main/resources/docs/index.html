Useless Auto Clicker (UAutoClicker) - это программа, предназначенная для автоматизации действий пользователя. Программа способна генерировать и захватывать события для большинства клавиш клавиатуры и мыши для большинства операционных систем. 
Функционал программы:

  • Генерация событий мыши и клавиатуры.Программа генерирует глобальные события для мыши и клавиатуры, так что данные события будут восприниматься любыми другими программами будто бы вы личто нажали нужную кнопку или переместили мышь;
  • Встроенный редактор скрипта.Все события записываются в удобном редакторе, поэтому написать сложное поведение не составит никаких проблем;
  • Поддержка JavaScript и Java синтаксиса и библиотек.Создать функцию, создать класс? Сложение, вычитание, посчитать логарифм и квадратный корень? Все включено;
  • Захват событий клавиатуры и мыши.Не нужно вручную писать сложные команды подбирая координаты пикселей. Выберите нужную опцию и быстро вставьте нужную команду в скрипт;
  • Захват всех событий по одной кнопке.Просто нажмите нужную кнопку и покажите программе что нужно делать;
  • Поддержка Windows, Linux, IOS.Работает практически везде, где работает Java 8;
  • Встроенный HTTP сервер.Позволит писать скрипты даже в браузере и на мобильном телефоне;
  • Встроенный Socket сервер.Более быстрый вариант для управлением машиной;
  • Работа с буфером обмена
  • Набор встроенных шаблоновЗабыли как называется нужный метод или лень писать все вручную? Просто выберите нужный из списка заготовленных шаблонов;

Проект быстро развивается и мы готовы выслушать любые пожелания и замечания. Пишите о найденных багах в темах с соответсвующей версией программы. Следите за обновлениями.
